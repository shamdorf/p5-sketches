🔀 Was sind Verzweigungen?

Mit einer Verzweigung kann das Programm unterschiedliche Wege gehen – je nachdem, ob eine Bedingung true oder false ist.

  • Das Schlüsselwort dafür ist if.
  • Zusätzlich kann man auch else und else if benutzen.

if (bedingung) {
  // dieser Block wird ausgeführt, wenn die Bedingung wahr ist
} else {
  // dieser Block wird ausgeführt, wenn die Bedingung falsch ist
}
🔎 Bedingungen in Processing

Eine Bedingung ist ein Ausdruck, der entweder true oder false ergibt.

  • Die Bedingung steht in Klammern ( ... )
  • Sie enthält oft einen Vergleichsoperator:

🔢 Vergleichsoperatoren

  • == → gleich
  • != → ungleich
  • > → größer als
  • < → kleiner als
  • >= → größer oder gleich
  • <= → kleiner oder gleich

📚 Beispiel


if (x > 100) {
  fill(255, 0, 0);
} else {
  fill(0, 255, 0);
}
🧪 Bedingungen kombinieren

Man kann Bedingungen auch miteinander kombinieren.

  • && → UND: Beide Bedingungen müssen true sein
  • || → ODER: Eine der Bedingungen muss true sein
  • ! → NICHT: Kehrt den Wahrheitswert um

📚 Beispiel


if (x > 100 && y < 50) {
  ellipse(x, y, 30, 30);
}
🎨 Beispiel: Farbwechsel mit if
🌈 Farbe abhängig von Position 

void draw() {
  if (mouseX < width / 2) {
    background(255, 0, 0); // rot
  } else {
    background(0, 0, 255); // blau
  }
}
📊 Mehrere Bedingungen prüfen
🚦 Farbwechsel mit if ... else if ... else 

void draw() {
  if (mouseX < 100) {
    background(255, 0, 0);  // rot
  } else if (mouseX < 200) {
    background(255, 255, 0); // gelb
  } else {
    background(0, 255, 0); // grün
  }
}
📌 Häufige Fehler bei if
  • = ist eine Zuweisung, == ist ein Vergleich!
  • Immer geschweifte Klammern { ... } benutzen.
  • Ein Block kann leer sein, sollte aber logisch Sinn ergeben.
🧠 Fragen zu Kapitel 3
Was macht eine if-Verzweigung?

Sie prüft eine Bedingung und führt Anweisungen aus, wenn diese erfüllt ist.

Was ist der Unterschied zwischen = und ==?

= weist einen Wert zu, == vergleicht zwei Werte.

Was passiert bei mehreren Bedingungen?

Du kannst else if und else nutzen, um weitere Bedingungen zu prüfen.

Wie prüft man, ob zwei Dinge gleichzeitig gelten?

Mit dem logischen UND &&.
