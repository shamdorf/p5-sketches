🔀 If-Anwesiungen
🔀 Was ist eine If-Anweisung (Verzweigung)

Mit einer Verzweigung kann das Programm unterschiedliche Wege gehen – je nachdem, ob eine Bedingung true oder false ist.

  • Das Schlüsselwort dafür ist if.
  • Zusätzlich kann man auch else und else if benutzen.

Syntax

if (Bedingung) {
   // dieser Block wird ausgeführt, wenn die Bedingung wahr ist
} else {
  // dieser Block wird ausgeführt, wenn die Bedingung falsch ist
}

Aufpassen bei der Syntax! Nach der Bedingung darf KEIN ";" (Semicolon) stehen. 

if (Bedingung); { ⛔ //SYNTAXFEHLER!
  ...
}

🔎 Bedingungen in Processing

Eine Bedingung ist ein Ausdruck, der entweder true oder false ergibt.

  • Die Bedingung steht in Klammern ( ... )
  • Sie enthält oft einen Vergleichsoperator:

🔢 Vergleichsoperatoren

Operator Erklärung
== gleich
!= ungleich
> größer als
< kleiner als
>= größer oder gleich
<= kleiner oder gleich
  • == ist ein Vergleichsoperator. = ist der Zuweisungsoperator um Variablenwerte zu ändern!
    if (zahl = 12) { ⛔ //Ist nicht korrekt!
  • Es heißt ">=" (größer gleich) und nicht "=>" (gleich größer)

📚 Beispiel

if (x > 100) {  //Wenn x größer als 100 ist, setze Farbe auf rot
  fill(255, 0, 0);
} else {  //Ansonsten setze Farbe auf grün
  fill(0, 255, 0);
}
Syntax für if-else, if-else-if

if-else

if (Bedingung) {
   // dieser Block wird ausgeführt, wenn die Bedingung wahr ist
} else {
  // dieser Block wird ausgeführt, wenn die Bedingung falsch ist
}

Syntax

if (Bedingung) {
   // dieser Block wird ausgeführt, wenn die Bedingung wahr ist
} else if (Bedingung2) {
   // dieser Block wird ausgeführt, wenn die zweite Bedingung wahr ist
} else if (Bedingung3) {
   // dieser Block wird ausgeführt, wenn die dritte Bedingung wahr ist
} else {
  // dieser Block wird ausgeführt, wenn alle Bedingungen falsch sind
}

🔀 If-Else-If
Motivation und Syntax

Mit else-if kann man mehrere Bedingungen abfragen und komplexere Abfragen aufbauen. Man kann so zum Beispiel testen, ob eine Zahl in einem bestimmten Bereich liegt.

Beispiel - In welchem Lebensabschnitt befindest du dich?

if (alter < 18) {
  println("Du bist noch minderjährig");
} else if (alter < 67) {
  println("Du bist erwachsen und im arbeitsfähigem Alter!");
} else {
  println("Du bist Rentner!");
}

Syntax

if (Bedingung) {
   // dieser Block wird ausgeführt, wenn die Bedingung wahr ist
} else if (Bedingung2) {
   // dieser Block wird ausgeführt, wenn die zweite Bedingung wahr ist
} else if (Bedingung3) {
   // dieser Block wird ausgeführt, wenn die dritte Bedingung wahr ist
} else {
  // dieser Block wird ausgeführt, wenn alle Bedingungen falsch sind
}

Aufpassen bei der Syntax! Nach den Bedingungen darf KEIN ";" (Semicolon) stehen. 

if (Bedingung); { //SYNTAXFEHLER!

🧩 Bedingungen kombinieren

Du kannst mehrere Bedingungen mit logischen Operatoren kombinieren:

  • && – UND: beide Bedingungen müssen zutreffen
  • || – ODER: eine der Bedingungen muss zutreffen
  • ! – NICHT: kehrt die Bedingung um

⚖️ Präzedenz (Reihenfolge der Auswertung)

  • ! wird zuerst ausgewertet
  • && kommt danach
  • || zuletzt

Um Missverständnisse zu vermeiden, solltest du Klammern setzen:
if ((x > 100 && y < 50) || !sichtbar)

📚 Beispiel: Farbe abhängig von Mausposition und Sichtbarkeit

Wenn sich die Maus rechts oben befindet oder die Variable sichtbar auf false gesetzt ist, wird der Hintergrund grün. Andernfalls ist er rot.
Mit der Taste kannst du sichtbar umschalten. 


boolean sichtbar = true;

void setup() {
  size(300, 200);
  textAlign(CENTER, CENTER);
  textSize(16);
}

void draw() {
  if ((mouseX > 100 && mouseY < 150) || !sichtbar) {
    background(100, 255, 100);
    text("Bedingung erfüllt", width / 2, height / 2);
  } else {
    background(255, 100, 100);
    text("Bedingung nicht erfüllt", width / 2, height / 2);
  }
}

void keyPressed() {
  sichtbar = !sichtbar;
}
🧪 Bedingungen kombinieren

Man kann Bedingungen auch miteinander kombinieren.

  • && → UND: Beide Bedingungen müssen true sein
  • || → ODER: Eine der Bedingungen muss true sein
  • ! → NICHT: Kehrt den Wahrheitswert um

📚 Beispiel


if (x > 100 && y < 50) {
  ellipse(x, y, 30, 30);
}

📊 Mehrere Bedingungen prüfen
🚦 Farbwechsel mit if ... else if ... else 

void draw() {
  if (mouseX < 100) {
    background(255, 0, 0);  // rot
  } else if (mouseX < 200) {
    background(255, 255, 0); // gelb
  } else {
    background(0, 255, 0); // grün
  }
}
📌 Häufige Fehler bei if
  • = ist eine Zuweisung, == ist ein Vergleich!
  • Immer geschweifte Klammern { ... } benutzen.
  • Ein Block kann leer sein, sollte aber logisch Sinn ergeben.
🧠 Fragen zu Kapitel 3
Was macht eine if-Verzweigung?

Sie prüft eine Bedingung und führt Anweisungen aus, wenn diese erfüllt ist.

Was ist der Unterschied zwischen = und ==?

= weist einen Wert zu, == vergleicht zwei Werte.

Was passiert bei mehreren Bedingungen?

Du kannst else if und else nutzen, um weitere Bedingungen zu prüfen.

Wie prüft man, ob zwei Dinge gleichzeitig gelten?

Mit dem logischen UND &&.
