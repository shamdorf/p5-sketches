Siehe Aufgabe 6.3b

Einführung in Arrays

Was sind Arrays?

Ein Array ist eine Datenstruktur, die es ermöglicht, mehrere Werte des gleichen Datentyps in einer einzigen Variablen zu speichern.

Man kann sich ein Array wie einen Becherhalter vorstellen, in dem mehrere Becher (Werte) Platz finden. Dies ist besonders nützlich, wenn man mit einer Vielzahl von Daten arbeitet, wie z.B. den Positionen von Objekten oder den Werten von Messungen.

Motivation - Warum sind Arrays sinnvoll?

Arrays sind besonders nützlich, wenn man mit großen Datenmengen arbeitet. Sie ermöglichen es, mehrere Werte effizient zu speichern und zu verwalten, ohne dass man für jeden Wert eine separate Variable erstellen muss. Dies führt zu übersichtlicherem und wartbarem Code.
Außerdem lassen sich Arrays mit Hilfe von Schleifen elegant und effizient verändern und auslesen.

Deklaration eines Arrays

int[] xarray = new int[5]; // Erzeugt ein Array mit 5 Integer-Elementen
  • xarray - Name des Arrays
  • [] - Eckigen Klammern zeigen an, dass es ein Array ist
  • new int [5] - Erstellt ein Integer-Array der Länge 5

Größe des Arrays

Die Größe eines Arrays kann mit der Eigenschaft length abgerufen werden: 

println(xarray.length); // Gibt die Anzahl der Elemente im Array aus

Dies ist nützlich, um sicherzustellen, dass man nicht versucht, auf ein nicht vorhandenes Element zuzugreifen.

Auf Elemente zugreifen

Jedes Element eines Arrays kann über seinen Index, der bei 0 beginnt, angesprochen werden. Zum Beispiel: 

println(xarray[0]); // Element 0 ausgeben
println(xarray[1]); // Element 1 ausgeben
println(xarray[2]);
println(xarray[3]);
println(xarray[4]); // letztes Element ausgeben

Um den Wert eines Elements zu ändern, kann man einfach eine Zuweisung vornehmen: 

xarray[3] = 42; // Setzt das vierte Element auf 42
Der Arrayindex beginnt bei 0, daher ist das erste Element xarray[0]. Wenn xarray die Länge 5 hat, kann man nicht auf xarray[5] zugreifen. Die Processing IDE wirft dann eine ArrayIndexOutOfBoundsException.
Übungsaufgaben 6.1a - 6.1c
Arrays und Schleifen
Array deklarieren und initialisieren

Ein Array wird in Processing/Java mit der folgenden Syntax deklariert: 

int[] fuenfer = new int[10]; // Erzeugt ein Array mit 10 Integer-Elementen

Initialisieren eines Arrays

Will man zum Beispiel alle Elemente des Arrays auf den Wert 5 setzen, kann man das zum Beispiel so machen: 

// Füllen ohne Schleife
int[] fuenfer = new int[10];
fuenfer[0] = 5;
fuenfer[1] = 5;
fuenfer[2] = 5;
fuenfer[3] = 5;
fuenfer[4] = 5;
fuenfer[5] = 5;
fuenfer[6] = 5;
fuenfer[7] = 5;
fuenfer[8] = 5;
fuenfer[9] = 5;
Bei vielen Elementen wird das schnell sehr unhandlich! Was macht ihr, wenn ihr 100, 1000 oder mehr Elemente initialisieren müsst?

Initialisieren mit Schleife

Eleganter und übersichtlicher geht es mit einer for-Schleife, die über alle Elemente des Arrays iteriert: 

// Füllen mit Schleife
int[] fuenfer = new int[10]; // Erzeugt ein Array mit 10 Integer-Elementen
for (int i = 0; i < 10; i++) {
    fuenfer[i] = 5;
}

Wir sehen, dass alles gleich bleibt, bis auf den Index (die Zahl in Klammern). Durch die Verwendung einer Schleife wird der Code kürzer und einfacher wartbar. Wenn Sie die Anzahl der Elemente ändern möchten, müssen Sie nur die Größe des Arrays anpassen.

Größe eines Arrays bestimmen

In der Schleife oben kann man die Zahl 10 durch fuenfer.length ersetzen. So passt sich die Schleife automatisch an die Größe des Arrays an: 

// Füllen mit Schleife
int[] fuenfer = new int[10];  // Erzeugt ein Array mit 10 Integer-Elementen
for (int i = 0; i < fuenfer.length; i++) {
    fuenfer[i] = 5;
}

Gegeben ist ein Integer Array der Länge 1000. Schreibe eine Schleife, um alle Werte des Arrays auf 42 zu setzen. Gebe anschließend das Array auf der Konsole aus. 

int[] foo = new int[1000]; // Erzeugt ein Array mit 1000 Integer-Elementen
Drücke hier für die Lösung:
Möglichkeit 1 - klappt in Processing
int[] foo = new int[1000]; // Erzeugt ein Array mit 1000 Integer-Elementen
for (int i = 0; i < 1000; i++) {
    foo[i] = 42;
}
println(foo); // Gibt das gesamte Array aus
Alternative, die auch in anderen Programmiersprachen klappt 
int[] foo = new int[1000]; // Erzeugt ein Array mit 1000 Integer-Elementen
for (int i = 0; i < 1000; i++) {
    foo[i] = 42;
    println(foo[i]); // Gibt jedes Element einzeln aus 
}
Beispiel: Bewegung von Bällen

Hier ist ein Beispiel, wie man drei Bälle mit normalen Variablen erstellt, die sich über den Bildschirm bewegen: 

int x1, x2, x3;

void setup() {
    size(400, 200);
    x1 = 0;
    x2 = 20;
    x3 = 40;
}

void draw() {
    background(255);
    ellipse(x1, 50, 10, 10);
    ellipse(x2, 50, 10, 10);
    ellipse(x3, 50, 10, 10);
    
    x1++;
    x2++;
    x3++;
}

Der Nachteil dieser Methode ist, dass man für jeden Ball eine separate Variable benötigt. Wenn man die Anzahl der Bälle erhöhen möchte, muss man den Code an mehreren Stellen ändern.

Eine bessere Lösung ist die Verwendung eines Arrays:

int[] xarray = new int[3];

void setup() {
    size(400, 200);
    for (int i = 0; i < xarray.length; i++) {
        xarray[i] = i * 20; // Berechne die x-Startposition
    }
}

void draw() {
    background(255);
    for (int i = 0; i < xarray.length; i++) {
        ellipse(xarray[i], 50, 10, 10); // Zeichnen
        xarray[i]++; // Hochzählen
    }
}
Übungsaufgaben 6.2
Eigenschaften in Arrays speichern
Personen repräsentieren

Um mehrere Personen mit verschiedenen Eigenschaften zu speichern, verwenden wir separate Arrays für jede Eigenschaft. Zum Beispiel können wir ein String-Array für Namen, ein int-Array für Alter und ein boolean-Array für das Geschlecht nutzen: 

String[] namen = { "Max Schmidt", "Lisa Marx", "Robin Meier" };
int[] alter = { 33, 28, 12 };
boolean[] weiblich = { false, true, false };

Der Index verbindet die Arrays: Index 0 steht für "Max Schmidt" und 33 Jahre, Index 1 für "Lisa Marx" und 28 Jahre, usw.

Um alle Personen mit ihren Eigenschaften auszugeben, können wir eine Schleife verwenden: 

for (int i = 0; i < namen.length; i++) {
    if (weiblich[i]) {
        println("Frau " + namen[i] + ", " + alter[i] + " Jahre alt");
    } else {
        println("Herr " + namen[i] + ", " + alter[i] + " Jahre alt");
    }
}

Gesamter Sketch für Personen repräsentieren

String[] namen = { "Max Schmidt", "Lisa Marx", "Robin Meier" };
int[] alter = { 33, 28, 12 };
boolean[] weiblich = { false, true, false };

for (int i = 0; i < namen.length; i++) {
    if (weiblich[i]) {
        println("Frau " + namen[i] + ", " + alter[i] + " Jahre alt");
    } else {
        println("Herr " + namen[i] + ", " + alter[i] + " Jahre alt");
    }
}

Konsolenausgabe

Herr Max Schmidt, 33 Jahre alt
Frau Lisa Marx, 28 Jahre alt
Herr Robin Meier, 12 Jahre alt

Grafische Objekte repräsentieren

Um mehrere grafische Objekte und deren Eigenschaften zu speichern, verwenden wir Arrays für die x- und y-Koordinaten. In diesem Beispiel speichern wir die Positionen von 100 Rechtecken in zwei Arrays: 

int[] xpos = new int[100];
int[] ypos = new int[100];

Wir initialisieren die Positionen der Rechtecke mit Zufallswerten: 

for (int i = 0; i < xpos.length; i++) {
    xpos[i] = (int)random(0, width);  // Zufallswert zwischen 0 und Breite
    ypos[i] = (int)random(0, height);  // (int) castet float zu int
}

Um die Rechtecke zu zeichnen, verwenden wir eine Schleife:

for (int i = 0; i < xpos.length; i++) {
    rect(xpos[i], ypos[i], 10, 10);  // Zeichnen
}

Zudem können wir die Positionen der Rechtecke mit den Cursortasten ändern und neue Zufallspositionen generieren, wenn die Leertaste gedrückt wird: 

void keyPressed() {
    if (keyCode == LEFT) {
        for (int i = 0; i < xpos.length; i++) {
            xpos[i]--;
        }
    }
    if (keyCode == RIGHT) {
        for (int i = 0; i < xpos.length; i++) {
            xpos[i]++;
        }
    }
    if (key == ' ') {
        for (int i = 0; i < xpos.length; i++) {
            xpos[i] = (int)random(0, width);
            ypos[i] = (int)random(0, height);
        }
    }
}

Gesamter Sketch für grafische Objekte repräsentieren

void setup() {
  size(400, 200);
  // Initialisiere die Positionen der Rechtecke
  for (int i = 0; i < xpos.length; i++) {
    xpos[i] = (int)random(0, width);
    ypos[i] = (int)random(0, height);
  }
}

void draw() {
  background(0); // Hintergrundfarbe
  fill(255); // Füllfarbe für die Rechtecke

  // Zeichne die Rechtecke
  for (int i = 0; i < xpos.length; i++) {
    rect(xpos[i], ypos[i], 10, 10);
  }
}

void keyPressed() {
  // Bewege alle Rechtecke nach links
  if (keyCode == LEFT) {
    for (int i = 0; i < xpos.length; i++) {
      xpos[i]--;
    }
  }

  // Bewege alle Rechtecke nach rechts
  if (keyCode == RIGHT) {
    for (int i = 0; i < xpos.length; i++) {
      xpos[i]++;
    }
  }

  // Setze neue zufällige Positionen für alle Rechtecke
  if (key == ' ') {
    for (int i = 0; i < xpos.length; i++) {
      xpos[i] = (int)random(0, width);
      ypos[i] = (int)random(0, height);
    }
  }
}
Übungsaufgaben 6.3
Objekte in Arrays speichern

Objekt-Arrays deklarieren

Sie können Objekte wie Vektoren oder Bilder in Arrays speichern, indem Sie den Klassennamen (z.B. PVector) als Typ verwenden und ihn mit eckigen Klammern versehen: 

PVector[] spaceships = new PVector[5];

Instanzieren der Objekte

Wichtig ist, dass das Array zunächst keine Objekte enthält, da noch kein new verwendet wurde. Wenn Sie versuchen, auf ein Element zuzugreifen, sehen Sie null: 

println(spaceships[0]); // Ausgabe: null

Ein Versuch, auf eine Eigenschaft eines nicht initialisierten Objekts zuzugreifen, führt zu einer NullPointerException: 

println(spaceships[0].x); // NullPointerException

Um Objekte im Array zu erstellen, müssen Sie durch das Array laufen und neue Objekte instanziieren. Zum Beispiel können Sie (0,0)-Vektoren verwenden: 

for (int i = 0; i < spaceships.length; i++) {
    spaceships[i] = new PVector();
}

Initialisieren der Vektoren

Jetzt sind die Arrayzellen mit Referenzen auf die erstellten Objekte gefüllt. Sie können auch Zufallswerte verwenden, um die Vektoren zu initialisieren: 

for (int i = 0; i < spaceships.length; i++) {
    float x = random(400);
    float y = random(400);
    spaceships[i] = new PVector(x, y);
}

Der Zugriff auf die einzelnen Zellen des Arrays erfolgt ebenfalls über die eckigen Klammern und den Index: 

for (int i = 0; i < spaceships.length; i++) {
    rect(spaceships[i].x, spaceships[i].y, 20, 20);
}

✅ Gesamter Sketch für Objekte in Arrays speichern

PVector[] spaceships = new PVector[5];  // Array für 5 PVector-Objekte

void setup() {
    size(400, 400);
    
    // Objekte im Array initialisieren
    for (int i = 0; i < spaceships.length; i++) {
        float x = random(400);
        float y = random(400);
        spaceships[i] = new PVector(x, y);
    }
}

void draw() {
    background(0);
    fill(255);
    
    // Rechtecke an den Positionen der Vektoren zeichnen
    for (int i = 0; i < spaceships.length; i++) {
        rect(spaceships[i].x, spaceships[i].y, 20, 20);
    }
}

void keyPressed() {  
  // Neue Zufallspositionen generieren
  for (int i = 0; i < spaceships.length; i++) {
      float x = random(400);
      float y = random(400);
      spaceships[i] = new PVector(x, y);
  }
    
}
Übungsaufgaben 6.4
Flexible Arrays

Nachteil von Arrays

  • Die Arraygröße ist fest
  • Sie muss bei der Deklaration angegeben werden und kann nicht verändert werden
  • Oft weiß man nicht, wie viele Variablen oder Objekte man insgesamt benötigt werden.

Wenn Sie während des Programmlaufs neue "Objekte" zu Ihrem Array hinzufügen möchten, müssen Sie eine Strategie finden.

Angenommen, Sie möchten sich Koordinatenpunkte merken. Sie speichern die x- und y-Koordinaten in zwei Arrays. Zu Beginn möchten Sie sich nur zwei Punkte merken, aber Sie erzeugen dennoch Arrays der Länge 100.

Lösung:

Die Grundidee ist, einen Array zu erzeugen, der wesentlich größer ist, als Sie zunächst brauchen, und nur einen Teil davon zu verwenden. 

int[] punktX = new int[100];
int[] punktY = new int[100];

void setup() {
  punktX[0] = 20;
  punktY[0] = 20;

  punktX[1] = 40;
  punktY[1] = 40;

  punktX[2] = 60;
  punktY[2] = 60;
}

Da Sie nicht wissen, wie viele Elemente dieser zwei Arrays wirklich "sinnvoll" verwendet werden, können Sie keine Schleife definieren, um die sinnvoll gefüllten Elemente für das Zeichnen zu verwenden. Die Lösung ist, eine neue Variable einzuführen, die angibt, wie weit das Array befüllt ist. In unserem Fall wäre die "Länge" gleich 3: 

int[] punktX = new int[100];
int[] punktY = new int[100];
int num = 3;

Jetzt können Sie eine Zeichenroutine einführen, die auf num basiert: 

void draw() {
  for (int i= 0; i < num; i++) {
    ellipse(punktX[i], punktY[i], 10, 10);
  }
}

Sie können zum Beispiel per Mausklick neue Kreise hinzufügen:

void mousePressed() {
  punktX[num] = int(random(0, width));
  punktY[num] = int(random(0, height));
  num++;
}

Beachten Sie, dass num immer eins höher ist als der aktuell verwendete Index, da es sich um die Länge des verwendeten Bereichs im Array handelt. Außerdem führt der obige Code zu einem Fehler, wenn mehr als 100 Punkte erreicht sind. Sie sollten Maßnahmen ergreifen, um diesen Fehler zu verhindern.

✅ Gesamter Sketch für Flexible Arrays

int[] punktX = new int[100];
int[] punktY = new int[100];
int num = 3;

void setup() {
    size(400, 400);
    
    // Initialisiere die Positionen der Punkte
    punktX[0] = 20;
    punktY[0] = 20;

    punktX[1] = 40;
    punktY[1] = 40;

    punktX[2] = 60;
    punktY[2] = 60;
}

void draw() {
    background(0); // Hintergrundfarbe
    fill(255); // Füllfarbe für die Punkte

    // Zeichne die Punkte
    for (int i = 0; i < num; i++) {
        ellipse(punktX[i], punktY[i], 10, 10);
    }
}

void mousePressed() {
    // Füge einen neuen Punkt hinzu
    punktX[num] = (int)random(0, width);
    punktY[num] = (int)random(0, height);
    num++;
}

Sie können auch Bälle wieder entfernen, indem Sie einfach num um eins verringern. Probieren Sie es aus, indem Sie beim Druck auf eine Tastaturtaste einen Ball entfernen. Auch hier könnte ein Fehler auftreten! Überlegen Sie, wann dieses Problem auftreten kann und wie Sie es vermeiden können.

Übungsaufgaben 6.5a
Zweidimensionale Arrays

In einem Array kann man wunderbar eine Reihe von Zahlen (oder Strings) speichern. Oft möchte man jedoch Zahlen (oder Strings) in einer Art Tabelle speichern. In der Mathematik nennt man das eine Matrix. Im Alltag kennen Sie Tabellenkalkulationsprogramme wie Excel, wo die Elemente mit zwei Indexzahlen angesprochen werden, z.B. mit Zeilennummer und Spaltennummer.

Beispiele für 2-dimensionale Strukturen:

  • Eine Tabelle, z.B. in einem Tabellenkalkulationsprogramm
  • Ein Spielfeld wie ein Schachbrett oder das Spielfeld bei PacMan
  • Ein Graustufen-Bild mit z.B. 2000x3000 Pixeln

Zu beachten ist, dass in einem 2-dimensionalen Array alle Elemente vom gleichen Typ sein müssen, also z.B. nur ganze Zahlen, nur Booleans oder nur Strings.

Einfache Arrays

Ein einfacher Zahlenarray wird wie folgt erstellt:

int[] a = new int[5];

Zu Beginn ist er mit Nullen gefüllt:

for (int i = 0; i < a.length; i++) {
  print(a[i] + " ");
}

Ausgabe: 0 0 0 0 0

Die zweite Dimension

Ein Array mit zwei Dimensionen wird so erzeugt:

int[][] b = new int[2][5];

Der 2-dimensionale Array hat zwei Indices. Die Länge des Arrays kann abgefragt werden: 

println(b.length); // Ausgabe: 2

Jedes Element enthält einen weiteren Array der Länge 5:

println(b[0].length); // Ausgabe: 5
println(b[1].length); // Ausgabe: 5

2-dimensionaler Array als Tabelle

Sie können auf einzelne Elemente mit folgender Notation zugreifen: 

b[0][0] = 4;
b[1][4] = 2;

Array mit Schleifen durchlaufen

Um alle Elemente zu durchlaufen, benötigen wir zwei verschachtelte Schleifen: 

for (int k = 0; k < b.length; k++) {
  for (int i = 0; i < b[k].length; i++) {
    print(b[k][i] + " ");
  }
  println();
}

Beispielausgabe:

4 0 0 0 0
0 0 0 0 2

Kurzschreibweise für das Initialisieren

Eine Kurzschreibweise für einen neuen Array mit Anfangswerten:

int[][] a = { {4, 0, 0, 0, 0}, {0, 0, 0, 0, 2} };

Beispiel: Tic Tac Toe

String[][] ticTacToe = {
  {"X", "O", "O"},
  {"-", "X", "-"},
  {"X", "O", "-"}
};

Beispiel: Auswahl

boolean[][] selection = {
  {false, true},
  {false, true},
  {true, false},
  {false, false}
};

Mehrdimensionale Arrays

Ja, es gibt auch 3- oder 4-dimensionale Arrays, allgemein als Tensoren bezeichnet. Ein Beispiel sind die Pixel in einem Bild, wo für jeden Pixel drei Farbwerte (RGB) definiert werden. 

int[][][] tensor = {
  {
    {255, 0},
    {0,   0}
  },
  {
    {0, 255},
    {255, 0}
  },
  {
    {0, 0  },
    {0, 255}
  }
};
Übungsaufgaben 6.6
Fragen zu Arrays
Was ist der Hauptvorteil von Arrays?

Arrays ermöglichen die Speicherung mehrerer Werte des gleichen Datentyps in einer einzigen Variable, was zu einem übersichtlicheren und wartbareren Code führt.

Wie greift man auf ein Element in einem Array zu?

Man greift auf ein Element in einem Array zu, indem man den Index des Elements in eckigen Klammern angibt, z.B. xarray[0] für das erste Element.

Was passiert, wenn man auf einen nicht existierenden Index zugreift?

Man erhält eine ArrayIndexOutOfBoundsException, da der Index außerhalb des gültigen Bereichs liegt.
