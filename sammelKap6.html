Arrays

Arrays sind Sammlungen von Elementen des gleichen Typs, die unter einem gemeinsamen Namen gespeichert werden. Sie ermöglichen den Zugriff auf mehrere Werte über einen Index.

Eigenschaften von Arrays

  • length: Gibt die Anzahl der Elemente im Array zurück.
  • indexOf(): Sucht nach einem Element und gibt dessen Index zurück.
  • push(): Fügt ein Element am Ende des Arrays hinzu.
  • pop(): Entfernt das letzte Element aus dem Array.
