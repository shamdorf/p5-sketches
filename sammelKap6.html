Einführung in Arrays

Was sind Arrays?

Ein Array ist eine Datenstruktur, die es ermöglicht, mehrere Werte des gleichen Datentyps in einer einzigen Variablen zu speichern.

Man kann sich ein Array wie einen Becherhalter vorstellen, in dem mehrere Becher (Werte) Platz finden. Dies ist besonders nützlich, wenn man mit einer Vielzahl von Daten arbeitet, wie z.B. den Positionen von Objekten oder den Werten von Messungen.

Motivation - Warum sind Arrays sinnvoll?

Arrays sind besonders nützlich, wenn man mit großen Datenmengen arbeitet. Sie ermöglichen es, mehrere Werte effizient zu speichern und zu verwalten, ohne dass man für jeden Wert eine separate Variable erstellen muss. Dies führt zu übersichtlicherem und wartbarem Code.
Außerdem lassen sich Arrays mit Hilfe von Schleifen elegant und effizient verändern und auslesen.

Deklaration eines Arrays

int[] xarray = new int[5]; // Erzeugt ein Array mit 5 Integer-Elementen
  • xarray - Name des Arrays
  • [] - Eckigen Klammern zeigen an, dass es ein Array ist
  • new int [5] - Erstellt ein Integer-Array der Länge 5

Größe des Arrays

Die Größe eines Arrays kann mit der Eigenschaft length abgerufen werden: 

println(xarray.length); // Gibt die Anzahl der Elemente im Array aus

Dies ist nützlich, um sicherzustellen, dass man nicht versucht, auf ein nicht vorhandenes Element zuzugreifen.

Auf Elemente zugreifen

Jedes Element eines Arrays kann über seinen Index, der bei 0 beginnt, angesprochen werden. Zum Beispiel: 

println(xarray[0]); // Element 0 ausgeben
println(xarray[1]); // Element 1 ausgeben
println(xarray[2]);
println(xarray[3]);
println(xarray[4]); // letztes Element ausgeben

Um den Wert eines Elements zu ändern, kann man einfach eine Zuweisung vornehmen: 

xarray[3] = 42; // Setzt das vierte Element auf 42
Der Arrayindex beginnt bei 0, daher ist das erste Element xarray[0]. Wenn xarray die Länge 5 hat, kann man nicht auf xarray[5] zugreifen. Die Processing IDE wirft dann eine ArrayIndexOutOfBoundsException.

Fragen zu Arrays
Was ist der Hauptvorteil von Arrays?

Arrays ermöglichen die Speicherung mehrerer Werte des gleichen Datentyps in einer einzigen Variable, was zu einem übersichtlicheren und wartbareren Code führt.

Wie greift man auf ein Element in einem Array zu?

Man greift auf ein Element in einem Array zu, indem man den Index des Elements in eckigen Klammern angibt, z.B. xarray[0] für das erste Element.

Was passiert, wenn man auf einen nicht existierenden Index zugreift?

Man erhält eine ArrayIndexOutOfBoundsException, da der Index außerhalb des gültigen Bereichs liegt.
