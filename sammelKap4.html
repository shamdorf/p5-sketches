Kapitel 4: Objekte in Processing

Einführung in Objekte

Objekte sind eine zentrale Komponente der objektorientierten Programmierung. Sie ermöglichen es, Daten und Funktionen zu kapseln, die zusammengehören. In Processing können wir eigene Klassen definieren, um Objekte zu erstellen.

Erstellen einer Klasse

Um eine Klasse in Processing zu erstellen, verwenden wir das Schlüsselwort class. Hier ist ein einfaches Beispiel für eine Klasse, die einen Punkt repräsentiert: 

class Punkt {
    float x;
    float y;

    Punkt(float x, float y) {
        this.x = x;
        this.y = y;
    }

    void anzeigen() {
        ellipse(x, y, 10, 10);
    }
}
Verwendung von Objekten

Nachdem wir eine Klasse erstellt haben, können wir Objekte dieser Klasse instanziieren. Hier ist ein Beispiel, wie man ein Objekt der Klasse Punkt verwendet: 

Punkt p = new Punkt(50, 50);
p.anzeigen();

Methoden in Klassen

Klassen können auch Methoden enthalten, die spezifische Funktionen für das Objekt bereitstellen. Hier ist ein Beispiel, das die Methode verschieben() hinzufügt: 

class Punkt {
    float x;
    float y;

    Punkt(float x, float y) {
        this.x = x;
        this.y = y;
    }

    void anzeigen() {
        ellipse(x, y, 10, 10);
    }

    void verschieben(float dx, float dy) {
        x += dx;
        y += dy;
    }
}

Zusammenfassung

Objekte sind ein leistungsfähiges Konzept in Processing, das es ermöglicht, komplexe Programme zu strukturieren und zu organisieren. Durch die Verwendung von Klassen und Methoden können wir unsere Programme modular und wiederverwendbar gestalten.
