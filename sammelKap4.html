Vektoren (PVector)

Vektoren werden in Processing verwendet, um den Ort und die Geschwindigkeit von Objekten zu definieren und zu verändern. Sie sind besonders nützlich in der Physik und Computergrafik, da sie sowohl Richtung als auch Größe darstellen können.

Eigenschaften von Vektoren

  • x: Die x-Koordinate des Vektors.
  • y: Die y-Koordinate des Vektors.
  • z: Die z-Koordinate des Vektors (optional, für 3D-Anwendungen).
  • magnitude(): Gibt die Länge des Vektors zurück.
  • normalize(): Wandelt den Vektor in einen Einheitsvektor um.
  • add(PVector v): Addiert einen anderen Vektor zu diesem Vektor.
  • sub(PVector v): Subtrahiert einen anderen Vektor von diesem Vektor.
  • mult(float n): Multipliziert den Vektor mit einem Skalar.
  • div(float n): Teilt den Vektor durch einen Skalar.

Strings in Processing

Die Klasse String in Processing ermöglicht die Arbeit mit Textdaten. Strings sind unveränderlich, was bedeutet, dass jede Modifikation einen neuen String erzeugt.

Übersicht über String-Methoden

  • length() - Gibt die Länge des Strings zurück.
  • charAt(index) - Gibt das Zeichen an der angegebenen Position zurück.
  • substring(start, end) - Gibt einen Teil des Strings zurück.
  • concat() - Verbindet zwei Strings.

String-Konkatenation

String text1 = "Hallo";
String text2 = "Welt";
String result = text1 + " " + text2; // "Hallo Welt"

Bilder in Processing einfügen

Um Bilder in Processing zu verwenden, müssen sie zuerst in das Projektverzeichnis geladen werden. Hier ist ein einfaches Beispiel: 

PImage img;

void setup() {
    img = loadImage("bild.png"); // Bild laden
    size(400, 400);
}

void draw() {
    image(img, 0, 0); // Bild anzeigen
}

Hintergrundmusik und Sounds in Processing

Hintergrundmusik einfügen

PApplet app;

void setup() {
    app = this;
    app.sound = new SoundFile(app, "musik.mp3");
    app.sound.loop(); // Musik im Hintergrund abspielen
}

Kurze Sounds abspielen

Um kurze Sounds nacheinander abzuspielen, kannst du die Methode play() verwenden. Hier ist ein Beispiel: 

SoundFile sound1;
SoundFile sound2;

void setup() {
    sound1 = new SoundFile(this, "schuss.mp3");
    sound2 = new SoundFile(this, "hupe.mp3");
}

void draw() {
    // Beispiel: Abspielen von Sounds bei Tastendruck
    if (keyPressed) {
        if (key == 's') {
            sound1.play(); // Schuss abspielen
        } else if (key == 'h') {
            sound2.play(); // Hupe abspielen
        }
    }
}
