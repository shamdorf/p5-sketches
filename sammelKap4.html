Primitive Datentypen vs Objekte
Primitive Datentypen

Primitive Datentypen sind die Datentypen die einfache Werte speichern wie ganze Zahlen, Kommazahlen, einzelne Zeichen oder Wahrheitswerte.

In Processing und vielen anderen Programmiersprachen gibt es folgende Datentypen:

Primitiver Datentyp Beispiel
int int z = 10; – ganze Zahl
float float y = 3.14; – Kommazahl
boolean boolean sichtbar = true; – wahr/falsch
char char taste = 'A'; – einzelnes Zeichen
Objekte

Jetzt wollen wir mehrere zusammengehörige Informationen in einem Container bündeln, z.B. den x- und y-Wert eines Objekts oder die unterschiedlichen Informationen zu einem Kunden (Vorname, Nachname, Mailadresse etc.). Hierfür gibt es in vielen Programmiersprachen sogenannte Objekte.

Ein Objekt ist eine Speicherstruktur, die sowohl Eigenschaften/Variablen als auch Aktionen/Methoden beinhaltet. Sie wird irgendwo im Speicher abgelegt und kann mehrere Daten speichern.

Wir haben schon den Objekt-Datentyp bzw. die KLasse String für Zeichenketten kennengelernt. Wie lernen in diesem Kapitel noch weitere Klassen wie PVector, PImage und Soundfilekennen.

In Kapitel 8 werden wir unsere eigenen Klassen für weitere Objekte erstellen.

Um ein Objekt einer Klasse zu erstellen muss man es wie eine Variable deklarieren und anschließend erzeugen. 

PVector vec;  //Deklariert einen Vektor
vec = new PVector(10, 20); // Erzeugt ein PVector-Objekt mit x=10 und y=20
Objekte

Jetzt möchte man häufig mehrere zusammengehörige Informationen in einem Container bündeln, z.B. den x- und y-Wert eines Vektors oder die unterschiedlichen Informationen zu einem Kunden (Vorname, Nachname, Mailadresse etc.).

In diesem Kapitel lernen Sie Objekte kennen, in denen man mehrere Informationen speichern kann. Ein Objekt ist ein Konstrukt irgendwo im Speicher, das mehrere Daten speichern kann.

Ein Objekt ist eine Speicherstruktur, die sowohl mehrere Eigenschaften (ähnlich zu Variablen) als auch Aktionen (ähnlich zu Funktionen) beinhaltet. Diese Aktionen werden auch Methoden genannt.

Was ist ein Vektor?

Ein Vektor im zweidimensionalen Raum ist einfach ein Pfeil in einem zweidimensionalen Koordinatensystem. Dabei ist es "egal", wo genau dieser Pfeil platziert ist. Wichtig ist nur, in welche Richtung er zeigt und wie lang er ist. Üblicherweise beschreibt man einen 2D-Vektor mit genau zwei Zahlen: die erste Zahl beschreibt den x-Anteil, die zweite Zahl den y-Anteil.

Objekt als Beispiel-Objekt

Ein Objekttyp wird in Java groß geschrieben. Beispiele sind PVector (nur Processing) und String. Ein Objekttyp ist immer auch eine Klasse. Eine Variable eines Objekttyps kann Objekte der entsprechenden Klasse speichern.

Methoden

Objekte können nicht nur mehr Informationen speichern als Variablen, sie bieten darüber hinaus auch Aktionen an, die etwas "tun", also etwas berechnen oder bewegen.

Vektoren (PVector)

Vektoren werden in Processing verwendet, um den Ort und die Geschwindigkeit von Objekten zu definieren und zu verändern. Sie sind besonders nützlich in der Physik und Computergrafik, da sie sowohl Richtung als auch Größe darstellen können.

Eigenschaften von Vektoren

  • x: Die x-Koordinate des Vektors.
  • y: Die y-Koordinate des Vektors.
  • z: Die z-Koordinate des Vektors (optional, für 3D-Anwendungen).
  • magnitude(): Gibt die Länge des Vektors zurück.
  • normalize(): Wandelt den Vektor in einen Einheitsvektor um.
  • add(PVector v): Addiert einen anderen Vektor zu diesem Vektor.
  • sub(PVector v): Subtrahiert einen anderen Vektor von diesem Vektor.
  • mult(float n): Multipliziert den Vektor mit einem Skalar.
  • div(float n): Teilt den Vektor durch einen Skalar.

🔢 Vektoren Beispiel: Ortsvektor und Geschwindigkeitsvektor

In diesem Beispiel wird ein Ball dargestellt, der sich basierend auf seiner Geschwindigkeit bewegt. Der Ortsvektor wird mit dem Geschwindigkeitsvektor addiert, um die Position des Balls zu aktualisieren. 


PVector position; // Ortsvektor
PVector velocity; // Geschwindigkeitsvektor

void setup() {
    size(800, 600);
    position = new PVector(width / 2, height / 2); // Setze den Anfangs-Ortsvektor in die Mitte des Canvas
    velocity = new PVector(2, 3); // Setze den Geschwindigkeitsvektor (x: 2, y: 3)
}

void draw() {
    background(255);

    // Aktualisiere die Position durch Addition des Geschwindigkeitsvektors
    position.add(velocity);

    // Überprüfe, ob der Ball den Rand des Canvas erreicht hat, und kehre die Richtung um
    if (position.x > width || position.x < 0) {
        velocity.x *= -1; // Umkehren der x-Richtung
    }
    if (position.y > height || position.y < 0) {
        velocity.y *= -1; // Umkehren der y-Richtung
    }

    // Zeichne den Ball
    fill(0);
    ellipse(position.x, position.y, 30, 30); // Zeichne den Ball mit dem aktuellen Ortsvektor
}

Strings in Processing

Die Klasse String in Processing ermöglicht die Arbeit mit Textdaten. Strings sind unveränderlich, was bedeutet, dass jede Modifikation einen neuen String erzeugt.

Übersicht über String-Methoden

  • length() - Gibt die Länge des Strings zurück.
  • charAt(index) - Gibt das Zeichen an der angegebenen Position zurück.
  • substring(start, end) - Gibt einen Teil des Strings zurück.
  • concat() - Verbindet zwei Strings.

String-Konkatenation

String text1 = "Hallo";
String text2 = "Welt";
String result = text1 + " " + text2; // "Hallo Welt"

Hintergrundmusik und Sounds in Processing

Um Hintergrundmusik und Sounds in Processing zu verwenden, musst du eine Sound-Library importieren. Wir benutzen im folgenden Beispiel die Library Sound von der Processing Foundation. Diese ermöglicht es, Sound-Dateien zu laden und abzuspielen.

Anleitung

Hintergrundmusik einfügen

import processing.sound.*;  //importieren der Sound-Library

SoundFile background;  //Deklarieren des SoundFile-Objekts background 

void setup() {
    background = new SoundFile(this, "music.mp3"); //Erzeugen des Objekts mit dem new-Operator
}

void draw() {
    background.loop(); // Hintergrundmusik abspielen
    // Hier kannst du weitere Zeichnungen oder Animationen hinzufügen
}

Kurze Sounds abspielen

Um kurze Sounds nacheinander abzuspielen, kannst du die Methode play() verwenden. Hier ist ein Beispiel: 

import processing.sound.*;  //importieren der Sound-Library

//Deklarieren der SoundFile-Objekte shoot und horn
SoundFile shoot;  
SoundFile horn;

void setup() {
    //Erzeugen der Objekte mit dem new-Operator
    shoot = new SoundFile(this, "shoot.mp3");
    horn = new SoundFile(this, "horn.mp3");
}

void draw() {
    // Beispiel: Abspielen von Sounds bei Tastendruck
    if (keyPressed) {
        if (key == 's') {
            shoot.play(); // Schuss abspielen
        } else if (key == 'h') {
            horn.play(); // Hupe abspielen
        }
    }
}
