HTML-Dokument

Ein HTML-Dokument ist der Grundbaustein jeder Webseite. Es besteht aus zwei Hauptteilen:

  • Head: Enthält Metainformationen über das Dokument, wie z.B. den Titel und Links zu CSS-Dateien.
  • Body: Beinhaltet den sichtbaren Inhalt der Webseite, einschließlich Text, Bilder und Links. 
<!DOCTYPE html>
<html lang="de">
    <head>
        <meta charset="UTF-8">
        <title>Titel der Seite</title>
    </head>
    <body>
        <h1>Willkommen</h1>
    </body>
</html>
HTML-Element

Ein HTML-Element besteht aus einem Start-Tag, optionalen Attributen, Inhalt und einem End-Tag: 

<p class="text">Dies ist ein Absatz.</p>
  • Start-Tag: <p>
  • Attribute: class="text"
  • Inhalt: Dies ist ein Absatz.
  • End-Tag: </p>
CSS-Regel

Eine CSS-Regel besteht aus einem Selektor und einer Deklaration: 

p {
    color: blue;
    font-size: 16px;
}
  • Selektor: p (wählt alle

    -Elemente aus)

  • Deklaration: { color: blue; font-size: 16px; }
Überschriften

HTML bietet sechs verschiedene Überschriftsebenen:

  • <h1> - Hauptüberschrift
  • <h2> - Unterüberschrift
  • <h3> - Weitere Unterüberschrift
  • <h4> - Vierte Ebene
  • <h5> - Fünfte Ebene
  • <h6> - Kleinste Überschrift
<h1>Dies ist eine Hauptüberschrift</h1>
Paragraphen

Text wird in HTML mit dem <p>-Tag eingefügt: 

<p>Dies ist ein Absatz.</p>

Um zusätzliche Leerzeichen oder Zeilenumbrüche zu erzeugen, können folgende Tags verwendet werden:

  • <br> für einen Zeilenumbruch
  • <hr> für eine horizontale Linie
<p>Erster Absatz.<br>Zweiter Absatz.</p>
Anordnen von Elementen

Es gibt verschiedene Methoden, um Elemente auf einer Webseite anzuordnen:

  • Display:
    • inline - Elemente nebeneinander, ohne Zeilenumbruch.
    • block - Elemente nehmen die gesamte Breite ein und beginnen auf einer neuen Zeile.
    • inline-block - Elemente nebeneinander, aber mit Breite und Höhe.
  • Float: Elemente können nach links oder rechts "geschwommen" werden.
  • Flexbox: Eine flexible Layout-Methode, die Elemente in einer Zeile oder Spalte anordnet.
  • Grid Layout: Ein zweidimensionales Layoutsystem, das Zeilen und Spalten verwendet.
Navigationsleiste

Das Grundgerüst einer Navigationsleiste könnte so aussehen:

<nav>
    <ul>
        <li><a href="#home">Home</a></li>
        <li><a href="#about">Über</a></li>
        <li><a href="#services">Dienste</a></li>
        <li><a href="#contact">Kontakt</a></li>
    </ul>
</nav>

Es gibt verschiedene Möglichkeiten zur Gestaltung der Navigation:

  • Horizontale Navigation
  • Vertikale Navigation
  • Dropdown-Menüs
Bilder und Hintergrundbilder

Um Bilder in HTML hinzuzufügen, wird das <img>-Tag verwendet: 

<img src="bild.jpg" alt="Beschreibung des Bildes">

Das alt-Attribut ist wichtig für Screenreader und beschreibt das Bild für Benutzer mit Sehbehinderungen.

Hintergrundbilder in CSS

Um ein Hintergrundbild in CSS zu setzen, wird die background-image-Eigenschaft verwendet: 

body {
    background-image: url('hintergrund.jpg');
    background-size: cover; /* Bild anpassen */
    background-repeat: no-repeat; /* Keine Wiederholung */
}
