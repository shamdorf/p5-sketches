Primitive Datentypen vs Objekte
Primitive Datentypen

Primitive Datentypen sind die Datentypen die einfache Werte speichern wie ganze Zahlen, Kommazahlen, einzelne Zeichen oder Wahrheitswerte.

Variable als einfachste Datenstruktur

  • Speichern genau einen Wert von einem Datentyp
  • Haben einen Namen
  • Wert kann sich bei Variablen zur Laufzeit ändern
  • Besitzen einen Gültigkeitsbereich (den sogenannten Scope)
    Scope einer Variable: der Anweisungsblock, in dem die Variable deklariert wurde, lokal oder global

In Processing und vielen anderen Programmiersprachen gibt es folgende Datentypen:

Primitiver Datentyp Beispiel
int int z = 10; – ganze Zahl
float float y = 3.14; – Kommazahl
boolean boolean sichtbar = true; – wahr/falsch
char char taste = 'A'; – einzelnes Zeichen
Objekte

Jetzt wollen wir mehrere zusammengehörige Informationen in einem Container bündeln, z.B. den x- und y-Wert eines Objekts oder die unterschiedlichen Informationen zu einem Kunden (Vorname, Nachname, Mailadresse etc.). Hierfür gibt es in vielen Programmiersprachen sogenannte Objekte.

Ein Objekt ist eine Speicherstruktur, die sowohl Eigenschaften/Variablen als auch Aktionen/Methoden beinhaltet. Sie wird irgendwo im Speicher abgelegt und kann mehrere Daten speichern.

Wir haben schon den Objekt-Datentyp bzw. die Klasse String für Zeichenketten kennengelernt. Wie lernen in diesem Kapitel noch weitere Klassen wie PVector, PImage und Soundfile kennen.

In Kapitel 8 werden wir unsere eigenen Klassen für weitere Objekte erstellen.

Bilder in Processing einfügen

Procesing unterstützt unterschiedliche Bildformate wie jpg, svg oder png.

Um Bilder in Processing zu verwenden, müssen wir im Projektverzeichnis (sketch-Ordner) erst einen Ordner "data" erstellen. In diesen Ordner können wir dann unsere Bild- und Sounddateien speichern.

Kostenlose Bilder auf pixaby

Was muss ich machen, um ein Bild in Processing zu benutzen?

  • Im Projektverzeichnis einen data-Ordner erstellen
  • Ein Bild herrunterladen und in den data-Ordner kopieren
  • Ein PImage-Objekt erstellen
  • Die Bilddatei in das Objekt laden
  • Das Bild zeichnen

Beispiel für ein Hintergrundbild

Download sonnensystem.jpg 
PImage img; //Erstellen eines PImage-Objekts mit dem Namen img
void setup() {
    size(800, 600);
    img = loadImage("sonnensystem.jpg"); // Laden des Bildes aus dem Ordner data
    
}

void draw() {
    image(img, 0, 0, width, height); // Bild an der Position (0, 0) mit größe (width, height) zeichnen
}

Beispiel für ein bewegtes Bild

Download deflin.png 
PImage img; // //Erstellen eines PImage-Objekts mit dem Namen img

void setup() {
  size(800, 600); // Fenstergröße
  img = loadImage("delfin.png"); // Laden des Bildes aus dem Ordner data
  img.resize(50, 50); // Befehl um die Bildgröße anzupassen
  noCursor(); // Mauszeiger ausblenden
}

void draw() {
  background(0, 0, 255); // Hintergrundfarbe
  image(img, mouseX - img.width / 2, mouseY - img.height / 2); // Bild zentriert auf die Mausposition zeichnen
}
Hintergrundmusik und Sounds in Processing

Processing unterstützt unter anderem die Sound-Formate wav und mp3.

Für das Abspielen müssen sich die Sound-Dateien wie bei den Bildern im Ordner data befinden.

Kostenlose Sounds auf pixaby

Was muss ich machen, um Sounds in Processing zu benutzen?

  • Im Projektverzeichnis einen data-Ordner erstellen
  • Einen Soundfile herrunterladen und in den data-Ordner kopieren
  • Eine Sound-Library in der Processing IDE installieren
  • Die Sound-Library im Sketch importieren
  • Ein Sound-Objekt erstellen
  • Den Sound in das Objekt laden
  • Den Sound abspielen
Anleitung zum Installieren der Sound-Library

Um Sounds in Processing zu verwenden, musst du eine Sound-Library importieren. Wir benutzen im folgenden Beispiel die Library Sound von der Processing Foundation. Diese ermöglicht es, Sound-Dateien zu laden und abzuspielen.

Beispiel - Hintergrundmusik einfügen

Um Musik einzufügen müssen wir die Musik-Datei in den data-Ordner speichern.

Download background.mp3 
import processing.sound.*;  //importieren der Sound-Library

SoundFile background;  //Deklarieren des SoundFile-Objekts background 

void setup() {
    background = new SoundFile(this, "music.mp3"); //Erzeugen des Objekts mit dem new-Operator
}

void draw() {
    background.loop(); // Hintergrundmusik abspielen
    // Hier kannst du weitere Zeichnungen oder Animationen hinzufügen
}

Beispiel - Kurze Sounds abspielen

Um kurze Sounds nacheinander abzuspielen, kannst du die Methode play() verwenden. Hier ist ein Beispiel: 

import processing.sound.*;  //importieren der Sound-Library

          
//Deklarieren der SoundFile-Objekte shoot und horn
SoundFile shoot;  
SoundFile horn;

void setup() {
    //Erzeugen der Objekte mit dem new-Operator
    shoot = new SoundFile(this, "shoot.mp3");
    horn = new SoundFile(this, "horn.mp3");
}

void draw() {
    // Beispiel: Abspielen von Sounds bei Tastendruck
    if (keyPressed) {
        if (key == 's') {
            shoot.play(); // Schuss abspielen
        } else if (key == 'h') {
            horn.play(); // Hupe abspielen
        }
    }
}
